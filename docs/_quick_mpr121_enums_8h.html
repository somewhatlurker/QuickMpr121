◆ mpr121AutoConfigBVA

enum mpr121AutoConfigBVA : uint8_t

possible auto-configuration "baseline value adjust" settings datasheet says this is the same as CL, but AN3899 is better here

Enumerator
MPR_AUTOCONFIG_BVA_DISABLED 

Baseline is not changed.

MPR_AUTOCONFIG_BVA_CLEAR 

Baseline is cleared.

MPR_AUTOCONFIG_BVA_SET_CLEAR3 

Baseline is set to the auto-config baseline with the lower 3 bits cleared.

MPR_AUTOCONFIG_BVA_SET_ALL 

Baseline is set to the auto-config baseline.

◆ mpr121ElectrodeConfigCL

enum mpr121ElectrodeConfigCL : uint8_t

possible "calibration lock" settings

Enumerator
MPR_CL_TRACKING_ENABLED 

Baseline tracking enabled, initial baseline value is current value in baseline value register.

MPR_CL_TRACKING_DISABLED 

Baseline tracking disabled.

MPR_CL_TRACKING_ENABLED_LOAD5 

Baseline tracking enabled, initial baseline value is loaded with the 5 high bits of the first 10-bit electrode data value.

MPR_CL_TRACKING_ENABLED_LOAD10 

Baseline tracking enabled, initial baseline value is loaded with all 10 bits of the first electrode data value.
